为最新版 VS Code 配置 HTML 开发环境，其实就是安装一个插件，再来个"一条龙"的设置。跟着下面这三步走，你就能拥有一个非常顺手、还能实时预览的开发环境了。

### 🚀 第1步：安装关键插件

VS Code 的丰富功能都来自插件，刚配置好环境，主要装这 3 个就够用了：

| 插件名称 | 主要作用 | 为什么推荐安装 |
| :--- | :--- | :--- |
| **Live Server** | **实时预览网页** | 这是你写 HTML 的核心工具。它会启动一个本地服务器，你每次保存代码，浏览器页面都会**自动刷新**，效果立竿见影，效率极高。 |
| **Prettier - Code formatter** | **自动格式化代码** | 帮你自动整理代码缩进和换行，让代码结构清晰、整洁。这在协作或阅读复杂代码时尤其有用，能帮你减少很多格式上的困扰。 |
| **Chinese (Simplified) Language Pack** | **中文界面** | 如果你对英文界面不太习惯，这个插件可以把 VSCode 的菜单、按钮等都变成简体中文，对新手非常友好。 |
| **Live Preview** | **内置预览** | 在 VS Code 的界面内直接看效果 |

**安装方法：**
点击 VS Code 左侧边栏的**扩展图标（或按 `Ctrl+Shift+X`）**，在搜索框里输入上面的插件名，找到后点击"**Install**"安装就行。

### ✍️ 第2步：创建并编写你的第一个网页

1.  **新建文件**：按 `Ctrl+N` （Mac 是 `Cmd+N`）新建一个文件。
2.  **保存为 HTML**：按 `Ctrl+S` （Mac 是 `Cmd+S`）保存，文件名写 `index.html`，**注意，后缀名 `.html` 非常重要**，务必要写上。
3.  **生成 HTML 骨架**：在新文件中，输入一个**英文感叹号 `!`** ，然后按键盘上的 **`Tab`** 键。神奇的事情会发生——VS Code 会自动为你生成一份完整的 HTML5 标准结构代码。
4.  **写点内容**：在生成好的 `<body>` 标签里，输入 `<h1>你好，世界！</h1>`，这就是你网页上的主要内容。

### ▶️ 第3步：运行和预览

*   **最佳方法 (Live Server)**：在 `index.html` 文件的编辑区域，**右键单击**，在菜单中选择 "**Open with Live Server**"。你的默认浏览器会自动打开并显示你的网页。之后你每次修改并保存代码，浏览器里的页面都会自动刷新，非常方便。
*   **备用方法 (直接打开)**：你也可以直接去电脑上找到 `index.html` 文件，**双击**用浏览器打开。不过这种方法不支持自动刷新，而且可能会遇到一些复杂的文件读取问题，所以**推荐优先使用 Live Server**。

### 💎 一些能让你效率翻倍的小技巧

1.  **开启自动保存**
    这个功能可以让你省去每次手动保存的麻烦。点击 VS Code 顶部菜单的 **"文件 (File)"**，然后勾选 **"自动保存 (Auto Save)"** 就好了。配合 Live Server，你每次敲击键盘，网页就实时更新了。

2.  **使用浏览器兼容性提示**
    如果你用的是 VS Code 1.100 或更高版本（现在的最新版肯定满足），当你把鼠标悬停在一些 HTML 标签或 CSS 属性上时，它会自动显示这项技术的**浏览器兼容性信息**，比如哪些浏览器支持、从哪个版本开始支持的。这对新手来说，是避免写出不兼容代码的绝佳助手。

### 🛠️ 补充方案：手动管理文件结构

当你的项目稍微复杂一点（比如要自己写 CSS 和 JS 文件），更好的做法是**先创建一个文件夹**来专门管理项目。
1.  在 VS Code 中点"**文件 (File)**" -> "**打开文件夹 (Open Folder)**"。
2.  选择一个你电脑上的位置，新建一个文件夹并选中它。
3.  然后在这个文件夹里，再创建 `index.html`、`style.css`、`script.js` 等文件，这样你的项目文件就不会散落一地了。

按照这个流程配置好，你的 VS Code 就成了一个趁手的 HTML 开发工具了。
