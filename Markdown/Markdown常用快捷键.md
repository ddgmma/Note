## Obsidian 常用快捷键（默认配置）

> 💡 提示：macOS 用户将 `Ctrl` 替换为 `Cmd`。

### 通用操作
| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + P` | 打开命令面板 |
| `Ctrl/Cmd + O` | 打开其他 Vault |
| `Ctrl/Cmd + ,` | 打开设置 |
| `Ctrl/Cmd + Shift + P` | 打开社区插件市场 |

### 笔记编辑 & 格式
| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + B` | 加粗选中文本 (`**text**`) |
| `Ctrl/Cmd + I` | 斜体 (`*text*`) |
| `Ctrl/Cmd + U` | 下划线（需启用内联格式）|
| `Ctrl/Cmd + Shift + S` | 删除线 (`~~text~~`) |
| `Ctrl/Cmd + K` | 插入链接 |
| `Ctrl/Cmd + L` | 聚焦文件名（快速重命名）|
| `Ctrl/Cmd + Enter` | 在内部链接后换行 |

### 插入块 & 元素
| 快捷键                    | 功能                    |
| ---------------------- | --------------------- |
| `` Ctrl/Cmd + ` ``     | 插入行内代码 (`` `code` ``) |
| `Ctrl/Cmd + Shift + V` | 切换源代码模式               |
| `/`（空行输入）              | 弹出块菜单（标题、列表、任务等）      |
| `Ctrl/Cmd + Alt + C`   | 插入代码块并选择语言            |

### 导航 & 视图
| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + Click` | 在新窗格中打开链接 |
| `Ctrl/Cmd + Tab` | 切换最近打开的笔记 |
| `Alt + ← / →` | 浏览历史（后退 / 前进）|
| `Ctrl/Cmd + \` | 切换侧边栏（文件浏览器）|
| `Ctrl/Cmd + Shift + G` | 打开图谱视图 |
| `Ctrl/Cmd + J` | 打开每日笔记（需启用插件）|

### 窗格（Pane）管理
| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + \` | 水平分割窗格 |
| `Ctrl/Cmd + Shift + \` | 垂直分割窗格 |
| `Ctrl/Cmd + W` | 关闭当前窗格 |
| `Ctrl/Cmd + Shift + T` | 重新打开最近关闭的笔记 |

### 任务与列表
| 快捷键 | 功能 |
|--------|------|
| `- [ ]` + 空格 | 创建可勾选任务 |
| `Tab` / `Shift + Tab` | 调整列表缩进层级 |

### 搜索
| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + F` | 当前笔记内搜索 |
| `Ctrl/Cmd + Shift + F` | 全局搜索（所有笔记）|
