# Linux 系统学习笔记

## 📁 文件与目录操作
| 命令 | 功能 | 示例 |
|------|------|------|
| `ls -la` | 列出所有文件（含隐藏文件） | `ls -la` |
| `cd <路径>` | 切换目录 | `cd /home/user` |
| `cd ..` | 返回上级目录 | `cd ..` |
| `cd ~` | 返回家目录 | `cd ~` |
| `pwd` | 显示当前目录路径 | `pwd` |
| `mkdir <目录名>` | 创建新目录 | `mkdir new_folder` |
| `rmdir <目录名>` | 删除空目录 | `rmdir empty_folder` |
| `cp <源文件> <目标>` | 复制文件 | `cp file1.txt file2.txt` |
| `mv <源文件> <目标>` | 移动或重命名文件 | `mv old.txt new.txt` |
| `rm <文件>` | 删除文件 | `rm file.txt` |
| `rm -rf <目录>` | 强制删除目录及内容 | `rm -rf folder/` |
| `find <路径> -name "<模式>"` | 查找文件 | `find /home -name "*.txt"` |
| `grep "<文本>" <文件>` | 在文件中搜索文本 | `grep "hello" file.txt` |
| `which <命令>` | 查找命令路径 | `which python` |

## 🔐 权限管理
| 命令 | 功能 | 示例 |
|------|------|------|
| `chmod <权限> <文件>` | 修改文件权限 | `chmod 755 script.sh` |
| `chown <用户:组> <文件>` | 修改文件所有者 | `chown user:group file.txt` |
| `sudo <命令>` | 以root权限执行命令 | `sudo apt update` |
| `su - <用户>` | 完全切换用户 | `su - username` |
| `whoami` | 查看当前用户 | `whoami` |
| `id` | 查看用户信息 | `id` |


## ⚙️ 系统管理
| 命令 | 功能 | 示例 |
|------|------|------|
| `ps aux` | 查看所有进程 | `ps aux` |
| `top` | 实时系统监控 | `top` |
| `df -h` | 查看磁盘空间 | `df -h` |
| `free -h` | 查看内存使用 | `free -h` |
| `uname -a` | 查看系统信息 | `uname -a` |
| `reboot` | 重启系统 | `sudo reboot` |
| `shutdown -h now` | 立即关机 | `sudo shutdown -h now` |
| `shutdown -r +10` | 10分钟后重启 | `sudo shutdown -r +10` |
| `kill <PID>` | 结束进程 | `kill 1234` |
| `kill -9 <PID>` | 强制结束进程 | `kill -9 1234` |
| `pkill <进程名>` | 按进程名结束 | `pkill process_name` |
| `bg` | 后台运行进程 | `bg` |
| `fg` | 前台恢复进程 | `fg` |


## 🌐 网络相关
| 命令 | 功能 | 示例 |
|------|------|------|
| `ping <主机>` | 测试网络连接 | `ping baidu.com` |
| `ifconfig` | 查看网络接口 | `ifconfig` |
| `netstat -tulpn` | 查看网络连接和端口 | `netstat -tulpn` |
| `ssh <用户@主机>` | 远程连接 | `ssh user@host` |
| `scp <文件> <用户@主机:路径>` | 安全复制文件 | `scp file.txt user@host:~/` |
| `scp -r <目录> <用户@主机:路径>` | 传输整个目录 | `scp -r folder/ user@host:~/` |


## 📦 包管理
### Ubuntu/Debian
| 命令 | 功能 | 示例 |
|------|------|------|
| `apt install <包名>` | 安装软件包 | `sudo apt install vim` |
| `apt remove <包名>` | 卸载软件包 | `sudo apt remove package` |
| `apt update` | 更新软件包列表 | `sudo apt update` |
| `apt upgrade` | 升级已安装的软件包 | `sudo apt upgrade` |
| `apt update && apt upgrade -y` | 更新并自动确认升级 | `sudo apt update && sudo apt upgrade -y` |

### RedHat/CentOS/Alibaba Linux
| 命令 | 功能 | 示例 |
|------|------|------|
| `yum install <包名>` | 安装软件包 | `sudo yum install vim` |
| `yum remove <包名>` | 卸载软件包 | `sudo yum remove package` |
| `yum update` | 更新所有软件包 | `sudo yum update` |
| `yum search <关键词>` | 搜索软件包 | `sudo yum search python` |


## 📦 压缩解压
| 命令 | 功能 | 示例 |
|------|------|------|
| `tar -czf <压缩包> <目录>` | 压缩目录 | `tar -czf backup.tar.gz folder/` |
| `tar -xzf <压缩包>` | 解压文件 | `tar -xzf backup.tar.gz` |
| `zip <压缩包> <文件>` | 创建zip压缩包 | `zip archive.zip file.txt` |
| `unzip <压缩包>` | 解压zip文件 | `unzip archive.zip` |


## 👥 用户管理
| 命令 | 功能 | 示例 |
|------|------|------|
| `useradd <用户名>` | 添加用户 | `sudo useradd newuser` |
| `passwd <用户名>` | 修改用户密码 | `sudo passwd username` |
| `userdel -r <用户名>` | 删除用户及家目录 | `sudo userdel -r username` |
| `usermod -aG <组> <用户>` | 将用户添加到组 | `usermod -aG sudo username` |


## 💻 Vim 配置
### Vim 基础配置
```vim
" 基本设置
syntax on                    " 启用语法高亮
set number                   " 显示绝对行号
set relativenumber           " 显示相对行号（当前行为0，上下为距离）

" 缩进设置
set expandtab                " 将 Tab 键转换为空格
set tabstop=4                " Tab 显示宽度为 4 个字符
set shiftwidth=4             " 自动缩进和 >> << 操作的缩进宽度为 4
set softtabstop=4            " Backspace 能一次删除 4 个空格（模拟 Tab 删除）
set autoindent               " 新行自动继承上一行的缩进
set smartindent              " 根据代码结构智能增加缩进（如遇到 if、{ 等）

" 其他设置
set cursorline               " 高亮显示当前光标所在行
set showmatch                " 输入 ) ] } 时短暂高亮匹配的左括号
set incsearch                " 输入搜索内容时即时高亮匹配项（增量搜索）
set hlsearch                 " 高亮显示所有匹配的搜索结果
```
