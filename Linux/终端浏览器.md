### 🚀 一键安装与快速启动

你可以直接在虚拟机终端里，执行下面这组命令来一次性安装所有浏览器（如果你用 Ubuntu/Debian 的话）：

bash

```
# 1. 更新软件包列表
sudo apt update

# 2. 一键安装全家桶（Browsh除外，它安装方式不同）
sudo apt install lynx links2 elinks w3m w3m-img -y

# 3. 安装Browsh（通过下载预编译二进制文件，最简单）
# 先进入你的下载目录，比如 ~/Downloads
cd ~/Downloads
# 下载最新版（请访问 https://www.brow.sh/docs/installation/ 确认最新版本号）
wget https://github.com/browsh-org/browsh/releases/download/v1.8.4/browsh_1.8.4_linux_amd64.deb
# 安装
sudo dpkg -i browsh_*.deb
# 如果出现依赖问题，运行以下命令修复
sudo apt --fix-broken install -y
```



安装完成后，可以通过以下命令快速启动它们：

bash

```
# 启动不同浏览器访问同一个网站（例如Hacker News），对比效果
lynx https://news.ycombinator.com
links2 https://news.ycombinator.com
elinks https://news.ycombinator.com
w3m https://news.ycombinator.com
browsh https://news.ycombinator.com
```

### 🎯 对比实验与实用技巧

为了让你直观感受差异，可以做这个**对比实验**：

1. 先用 `lynx` 和 `links2` 分别打开 `https://text.npr.org`（这是NPR专为文本浏览器优化的纯文字版新闻站），感受它们的“本职工作”。
2. 再用 `w3m` 打开一张图片的URL，比如 `https://example.com/image.jpg`，看看字符画效果。
3. 最后，用 `browsh` 挑战一下 `https://gmail.com` 或 `https://twitter.com`，观察它如何处理现代网页。

**通用操作技巧**：

- **导航**：通常使用**方向键**，`Tab`键跳转链接。
- **退出**：绝大多数按 **`q`** 然后确认，或 `Shift+Q`。
- **帮助**：在浏览器内按 **`?`** 或 **`H`**。
- **刷新**：通常按 **`Ctrl+R`**。

### 💡 一些有趣的发现

你可能还会发现，这些工具除了浏览网页，还有一些“隐藏技能”：

- **检查本地HTML**：`lynx ./本地文件.html` 可以直接预览你写的HTML页面效果。
- **脚本好帮手**：它们可以用来写爬虫或自动化脚本，稳定地获取网页内容。
- **复古BBS体验**：用 `telnet` 或 `ssh` 连接一些传统的文字BBS站，配合这些浏览器，能体验到1980年代的“赛博冲浪”感。
