## Claude Code 高频命令速查表

### ⌨️ 最常用快捷键

| 快捷键        | 功能                               |
| ------------- | ---------------------------------- |
| `Ctrl + C`    | 取消当前操作                       |
| `Ctrl + D`    | 退出会话                           |
| `Ctrl + L`    | 清屏（保留历史）                   |
| `Esc + Esc`   | 撤销 AI 最后一次代码改动（救命键） |
| `Shift + Tab` | 切换权限模式（normal/auto/plan）   |

### 📋 最常用斜杠命令

| 命令       | 功能                                 |
| ---------- | ------------------------------------ |
| `/help`    | 查看帮助                             |
| `/clear`   | 清空对话历史                         |
| `/init`    | 扫描项目生成 CLAUDE.md（新项目首推） |
| `/cost`    | 查看 Token 用量和费用                |
| `/model`   | 切换模型                             |
| `/compact` | 压缩对话上下文                       |

### 🔤 快速前缀

| 前缀 | 功能           | 示例                    |
| ---- | -------------- | ----------------------- |
| `/`  | 执行斜杠命令   | `/help`                 |
| `!`  | 执行 bash 命令 | `! git status`          |
| `@`  | 引用文件       | `@src/main.py 解释这个` |

### 🖥️ CLI 命令

| 命令              | 功能             |
| ----------------- | ---------------- |
| `claude`          | 启动会话         |
| `claude "提示词"` | 带提示启动       |
| `claude -c`       | 继续上次对话     |
| `claude update`   | 更新 Claude Code |

---

### 💡 核心技巧

- **新项目先跑 `/init`** → 让 AI 记住项目结构
- **改坏了按 `Esc + Esc`** → 立即回滚
- **不确认代码时用 `Shift + Tab` 切到 `plan` 模式** → 只出方案不改代码
