# 🦞 OpenClaw 斜杠命令速查手册

> **适用版本：** OpenClaw（龙虾）
> **用途：** 在聊天框中输入命令，控制 OpenClaw 的行为

---

## 一、📖 帮助与信息

| 命令        | 功能                                             | 示例        |
| :---------- | :----------------------------------------------- | :---------- |
| `/help`     | 显示帮助信息                                     | `/help`     |
| `/commands` | 列出所有可用斜杠命令                             | `/commands` |
| `/status`   | 查看 OpenClaw 当前运行状态（模型、版本、连接等） | `/status`   |

---

## 二、🌐 网关/渠道连接

| 命令               | 功能                                                    | 示例              |
| :----------------- | :------------------------------------------------------ | :---------------- |
| `/gateway-status`  | 查看所有已连接的渠道状态（Telegram/WhatsApp等是否在线） | `/gateway-status` |
| `/gwstatus`        | 同上，简写版                                            | `/gwstatus`       |
| `/auth [provider]` | 重新认证某个 AI 提供商的 API Key                        | `/auth openai`    |

---

## 三、🤖 Agent（智能体）管理

| 命令                    | 功能                              | 示例                  |
| :---------------------- | :-------------------------------- | :-------------------- |
| `/agents`               | 查看所有已创建的 Agent 列表       | `/agents`             |
| `/agent <id>`           | 切换到指定 ID 的 Agent            | `/agent my-assistant` |
| `/crestodian [request]` | **钳卫机制** — 风险评估与操作监督 | `/crestodian`         |

> 💡 Agent = 你可以创建多个不同性格/配置的 AI 助手，随时切换。

---

## 四、💾 会话管理

| 命令             | 功能             | 示例                 |
| :--------------- | :--------------- | :------------------- |
| `/sessions`      | 查看所有会话列表 | `/sessions`          |
| `/session <key>` | 切换到指定会话   | `/session work-chat` |

> 💡 会话 = 按主题/任务划分的对话记录，方便分类管理。

---

## 五、🧠 模型切换

| 命令                      | 功能             | 示例                               |
| :------------------------ | :--------------- | :--------------------------------- |
| `/models`                 | 查看可用模型列表 | `/models`                          |
| `/model <provider/model>` | 切换到指定模型   | `/model openai/gpt-4o`             |
|                           |                  | `/model anthropic/claude-sonnet-4` |
|                           |                  | `/model google/gemini-2.0-flash`   |

> 💡 格式：`厂商名/模型名`，前提是你配置了该厂商的 API Key。

---

## 六、🎯 推理深度控制

| 命令             | 功能     | 说明         |
| :--------------- | :------- | :----------- |
| `/think off`     | 关闭推理 | 最快但最浅   |
| `/think minimal` | 最低推理 | 简单问题用   |
| `/think low`     | 低级推理 | ⭐ 日常推荐   |
| `/think medium`  | 中级推理 | ⭐ 日常推荐   |
| `/think high`    | 高级推理 | 复杂问题用   |
| `/think xhigh`   | 极高推理 | 深度分析用   |
| `/think max`     | 最大推理 | 最慢但最深入 |

> 💡 推理深度越高 → 回答越深入，但响应越慢、Token 消耗越多。

---

## 七、⚡ 快速模式

| 命令           | 功能                                   | 示例           |
| :------------- | :------------------------------------- | :------------- |
| `/fast status` | 查看当前快速模式状态                   | `/fast status` |
| `/fast on`     | **开启**快速模式（跳过推理，轻量回复） | `/fast on`     |
| `/fast off`    | **关闭**快速模式（恢复完整推理）       | `/fast off`    |

> 💡 适合：问时间、天气、简单计算等不需要深度思考的场景。

---

## 八、🔍 调试与日志

| 命令             | 功能                                       | 示例             |
| :--------------- | :----------------------------------------- | :--------------- |
| `/verbose on`    | 开启详细日志模式（显示更多内部信息）       | `/verbose on`    |
| `/verbose off`   | 关闭详细日志                               | `/verbose off`   |
| `/trace on`      | 开启追踪模式（显示内部调用链）             | `/trace on`      |
| `/trace off`     | 关闭追踪                                   | `/trace off`     |
| `/reasoning on`  | **显示 AI 的思考过程**（Chain of Thought） | `/reasoning on`  |
| `/reasoning off` | 隐藏思考过程                               | `/reasoning off` |

---

## 九、💰 Token 用量统计

| 命令            | 功能                           |
| :-------------- | :----------------------------- |
| `/usage off`    | 不显示用量信息                 |
| `/usage tokens` | 只显示 Token 消耗数量          |
| `/usage full`   | 显示详细用量信息（含费用估算） |

## 十、🔐 权限与安全

| 命令               | 功能                                         | 说明               |
| :----------------- | :------------------------------------------- | :----------------- |
| `/elevated status` | 查看当前提权状态                             |                    |
| `/elevated on`     | **完全提权** — AI 可自由执行敏感操作         | ⚠️ 风险较高         |
| `/elevated off`    | **关闭提权** — AI 不能执行敏感操作           | 🔒 最安全           |
| `/elevated ask`    | **每次询问** — AI 执行敏感操作前征求你的同意 | ✅ **推荐日常使用** |
| `/elevated full`   | **完全信任** — 等同于 `on`                   | ⚠️                  |
| `/elev`            | 同上，简写版                                 | `/elev on`         |

> 🔒 **强烈建议新手保持 `ask` 模式**，防止 AI 误操作。

---

## 十一、🔄 对话控制

| 命令                  | 功能                                  | 示例                  |
| :-------------------- | :------------------------------------ | :-------------------- |
| `/new`                | **清除当前对话历史**，开启全新对话    | `/new`                |
| `/reset`              | 同上，等效命令                        | `/reset`              |
| `/abort`              | **立即终止** AI 正在执行的任务或回复  | `/abort`              |
| `/activation mention` | **仅 @提及 时** AI 才回应（群聊中用） | `/activation mention` |
| `/activation always`  | **所有消息** AI 都回应                | `/activation always`  |

---

## 十二、⚙️ 系统设置

| 命令        | 功能                           | 示例        |
| :---------- | :----------------------------- | :---------- |
| `/settings` | 打开设置菜单（交互式配置界面） | `/settings` |
| `/exit`     | **退出**当前会话或断开连接     | `/exit`     |

## 📋 快速索引（按使用频率）

```
🟢 最常用
  /new              — 重新开始对话
  /abort            — 喊停
  /model ...        — 切换模型
  /think low|medium — 控制推理深度
  /fast on|off      — 快速模式

🟡 偶尔用
  /status           — 查看状态
  /gwstatus         — 查看渠道连接
  /reasoning on     — 看 AI 在想什么
  /elevated ask     — 安全设置
  /usage tokens     — 看花了多少 Token

🔵 调试用
  /verbose on
  /trace on
  /commands
  /help

🔴 安全提醒
  默认保持 /elevated ask，不要随意开 full！
  恢复所有默认设置（终端运行）：
  openclaw settings reset
  或者，重新运行初始化向导：
  openclaw onboard
```
