## 📁 目录结构说明

### 文件夹

| 文件夹          | 说明                                                         |
| --------------- | ------------------------------------------------------------ |
| **`.git`**      | Git 版本控制文件夹（不用管它，OpenClaw 自带的）              |
| **`.openclaw`** | 🧠 **龙虾的核心目录**，里面存了配置、状态、日志、插件等，**不要乱删** |

---

### 核心配置文件（Markdown 文件）

这些文件都是**用大白话写的配置文件**，你可以直接用记事本打开看，甚至修改！

| 文件               | 中文名         | 是干什么用的                                               |
| ------------------ | -------------- | ---------------------------------------------------------- |
| **`IDENTITY.md`**  | 🆔 **身份文件** | 定义龙虾的"人设"——它的名字、性格、说话风格等               |
| **`SOUL.md`**      | 💖 **灵魂文件** | 定义龙虾的"核心价值观"——它的行为原则、注意事项             |
| **`AGENTS.md`**    | 🤖 **代理文件** | 定义龙虾可以调用哪些 AI 模型/工具（你的 API Key 配置在这） |
| **`TOOLS.md`**     | 🛠️ **工具文件** | 定义龙虾能用的所有技能（Skill）和钩子（Hook）              |
| **`BOOTSTRAP.md`** | 🚀 **启动文件** | 每次龙虾启动时自动执行的指令（就是 boot-md hook 用的）     |
| **`HEARTBEAT.md`** | 💓 **心跳文件** | 定时任务配置，让龙虾定期做某些事（比如每天早上汇报天气）   |
| **`USER.md`**      | 👤 **用户文件** | 存储关于你的信息——你的名字、偏好、习惯等，让龙虾更懂你     |

每次安装openclaw后**必做**：打开根目录，找到SOUL文件，添加如下内容：

```markdown
## 安全规则

### ⚡ 总原则

- **默认只有搜索和读取文件的权限**
- **修改、删除、复制、移动、重命名、写入任何文件，必须先征求我的同意，等我点头才能执行**

### 🛡️ 文件保护

- 删除、覆盖、移动、重命名任何文件前，必须先列出文件路径让我确认
- 绝对禁止修改以下目录：
  - `C:\Windows\`
  - `C:\Program Files\`
  - `C:\Program Files (x86)\`

### 💥 防系统崩溃

- 禁止执行注册表操作（`reg`、`regedit`、`reg add`）
- 禁止修改系统服务或计划任务
- 禁止关闭或禁用杀毒软件/防火墙/UAC

### 🔒 防信息泄露

- 禁止读取以下敏感内容：
  - `C:\Users\*\AppData\Local\` 下的任何文件（浏览器密码等）
  - 任何 `*.key`、`*.pem`、`*.env`、`*password*` 文件
  - SSH 密钥相关文件

### ✅ 明确允许的

- 读写我指定的项目文件夹，（如我手动在桌面创建了一个"test"文件夹，并告知了你。没有明确告知你，你就没有此文件夹的任何权限）
- 读取文档、代码文件等日常工作内容
```
